# CourierNett WebAPP

Webová/PWA verze osobní kalkulačky pro kurýry.

## Soubory

- `index.html` - hlavní stránka aplikace
- `styles.css` - vizuální styl
- `app.js` - aplikační logika
- `manifest.webmanifest` - PWA manifest
- `sw.js` - service worker
- `icon.svg` - ikona aplikace

## Spuštění

Aplikace je statická a nepoužívá backend. Pro běžné použití stačí otevřít `index.html` v prohlížeči. Pro testování PWA funkcí je lepší spustit složku přes lokální server, protože service worker v prohlížeči vyžaduje bezpečný kontext nebo localhost.
